3.1.3 Special inputs

Special inputs contain either complex formulas or macros. • Start of Production date (SoP) After selecting the input cell, a window will pop up. Note: When using dual-monitor setup, the pop-up window will usually appear on the main monitor, which doesn't need to be the one where the excel tool is shown. Do not be alarmed, excel is not frozen in that case.
