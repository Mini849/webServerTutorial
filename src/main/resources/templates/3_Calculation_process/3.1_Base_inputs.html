3.1 Base inputs

Base_Inputs sheet is used for setting up the calculation scenario and inputs. Based on the selected approach and calculation options, only necessary input rows will be shown. The calculation scenario and approach need to be known first, based on the decision proper selection has to be carried out. Afterwards, the PM or Sales representative need to set up their part of the inputs. Their inputs will influence not only what is available for other departments to fill in, but also the general appearance of all other calculation sheets. This section will not go into detail on specific inputs in the Base_Inputs sheet as they are described in the tool itself and in following sections of this document where needed. Exception are inputs which are macro enabled.
